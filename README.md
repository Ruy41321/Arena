# Makhia - UE5 C++ Project

*Makhìa is a project developed in **Unreal Engine 5** that implements the foundations of an Action RPG.  
The main focus of the project is on creating a solid, scalable, and fully multiplayer Combat System architecture based.  
It has been used the **Gameplay Ability System (GAS)** to implement a scalable multiplayer  
and the **Epic Online Services API (EOS)** to integrate a **Peer-to-Peer** Lobby System.*

For a deep dive into the architecture, please refer to the [Technical Documentation](#technical-documentation) below.

## 🚧 Project Status: In Development
This repository is a **public mirror** of my private development environment. It is specifically curated to showcase the project's architecture, C++ implementation, and technical progress. [more info below](#why-a-public-mirror)

## 🎯 Project Goals
- **Primary Goal:** Develop a fully functional Multiplayer 1v1 Game with an engaging melee RPG Combat System.
- **Secondary Goals:** Do practice, implement a robust and scalable codebase, maintain high performance standards, and build a fully scalable architecture to allow a future bigger project to start from here.

## Why a Public Mirror?
I have chosen to maintain a separate public repository for the following reasons:
- **Intellectual Property & Licensing:** The private repository contains third-party assets (from Fab/Marketplace) that cannot be redistributed publicly due to licensing restrictions.
- **Portfolio Focus:** This repo highlights the **Source Code**, making it easier for recruiters and developers to review my work without downloading gigabytes of binary data.
- **Clean Architecture:** Only core logic and documentation are pushed here, ensuring a focused look at the C++ systems.

## Technologies Used
- **Game Engine:** Unreal Engine 5.7
- **Languages:** C++, Blueprint Visual Scripting
- **Core Plugins:** Gameplay Ability System (GAS), Enhanced Input, Epic Integration Kit (EIK from Betide Studios)
- **Core Architecture:** Component-based Architecture, State Pattern, MVC (Model-View-Controller) Pattern

## Development Team
- **Technical Development:** Luigi (Solo developer for the entire codebase, architecture, game & combat systems, and C++/Blueprint integration).
- **Art/Audio Assets:** Visual and audio resources are entirely sourced from open-source material and external asset packages. Only a few resource have been done by me (mainly some VFX and animations) but the majority of the assets are from external sources.
