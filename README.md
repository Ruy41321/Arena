# Arena - UE5 C++ Project

Welcome to the public repository of **Arena**, a game currently in active development using **Unreal Engine 5** and **C++**.

## 🚧 Project Status: In Development
This repository is a **public mirror** of my private development environment. It is specifically curated to showcase the project's architecture, C++ implementation, and technical progress.

## 🎯 Project Goals
- **Primary Goal:** Develop a fully functional multiplayer 1v1 game with an engaging melee RPG Combat system.
- **Secondary Goals:** Do practice, implement a robust and scalable codebase, maintain high performance standards, and build a fully scalable architecture to allow a future bigger project to start from here.

## 📂 Why a Public Mirror?
I have chosen to maintain a separate public repository for the following reasons:
- **Intellectual Property & Licensing:** The private repository contains third-party assets (from Fab/Marketplace) that cannot be redistributed publicly due to licensing restrictions.
- **Portfolio Focus:** This repo highlights the **Source Code**, making it easier for recruiters and developers to review my work without downloading gigabytes of binary data.
- **Clean Architecture:** Only core logic and documentation are pushed here, ensuring a focused look at the C++ systems.

## 🛠 Technical Highlights
- **Engine:** Unreal Engine 5.7
- **Language:** C++ (Primary logic) & Blueprints (UI)
- **Features included in this repo:**
  - `Source/`: Full C++ classes and headers.
  - `Documentation/`: Technical breakdowns.
